NEAT¶

NEAT is a tool to catch the bad guys.

  • We use kdb+, q, python, flask, pandas, blaze, javascript, react, react-bootstrap, react-router, highcharts, underscore, c++
  • We encourage everyone to learn as much of the stack as possible. It makes the team more robust, but also it’s just more awesome that way.
  • Great overview of q
  • More q and kdb+ resources
  • We use react.js on the front end, start here and see here
  • We also use react-bootstrap, react-router, underscore, highcharts, and occasionally jquery and d3, in that order of frequency/importance.
  • In python land we use flask, pandas, and blaze.
  • If you can be comfortable with react, somewhat comfortable with flask, more comfortable with pandas, and a little conversant in q, that’s a great place to start.
  • Everything else is icing on the cake, but you’ll probably pick it up on the job.
