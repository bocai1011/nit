neat.test(verbose=False, junitfile=None, exit=False)¶

Runs the full neat test suite, outputting the results of the tests to sys.stdout.

This uses py.test to discover which tests to run, and runs tests in any ‘tests’ subdirectory within the neat module.

Parameters:

verbose : int, optional

Value 0 prints very little, 1 prints a little bit, and 2 prints the test names while testing.

junitfile : string, optional

If provided, writes the test results to an junit xml style xml file. This is useful for running the tests in a CI server such as Jenkins.

exit : bool, optional

If True, the function will call sys.exit with an error code after the tests are finished.
