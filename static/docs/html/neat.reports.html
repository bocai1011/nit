class neat.reports.reports.Bar(title, query, name=None, source=None, exports=None, cache=True, deps=None, link=None, widgets=None, grouper=None, field=None, summary=None, **kwargs)¶

Bases: neat.reports.reports.Graph

This is the python class for the Bar Chart widget.

In addition to the standard arguments, this widget takes an argument ‘stacking’, which is passed to the JS widget. The options for ‘stacking’ are: ‘normal’, ‘percent’, and None. The default is None.

‘normal’: render a stacked bar chart with no scaling ‘percent’: render a stacked bar chart where each group has been scalled to 1. Values are given as percentages None: render the bar chart without stacking. Bars which share a group will be shown beside one another.
